Celsius to Fahrenheit Open on JetBrains Academy

Write a program that reads a temperature in Celsius (°C) and shows its equivalent in Fahrenheit temperature (°F).

Use the following formula:

F = C * 1.8 + 32

Where C is a temperature in Celsius and F is a temperature in Fahrenheit.


Sample Input:
32.9
Sample Output:
91.22

Sample Input:
0
Sample Output:
32.0


Memory limit: 256 MB
Time limit: 8 seconds
